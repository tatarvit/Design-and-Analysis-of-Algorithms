## Завдання 1 — Максимальний потік у логістичній мережі

Реалізовано модель логістичної мережі, де товари рухаються від терміналів через склади до магазинів.

### Що було зроблено:
- побудовано граф логістичної мережі;
- вершини графа відповідають терміналам, складам і магазинам;
- ребра мають пропускну здатність;
- використано алгоритм Едмондса-Карпа;
- знайдено максимальний потік у мережі;
- сформовано таблицю фактичних потоків до магазинів.

### Логіка:
- термінали є джерелами товару;
- склади є проміжними вузлами;
- магазини є кінцевими отримувачами;
- алгоритм шукає доступні шляхи для збільшення потоку;
- процес триває, поки нові шляхи більше не знаходяться.

### Результат:
Було визначено:
- загальний максимальний потік;
- скільки товару отримав кожен магазин;
- які маршрути мають найменшу пропускну здатність;
- де знаходяться вузькі місця мережі.
